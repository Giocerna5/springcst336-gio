Programming Language(C++)

General Information Pros Cons Who uses c++?

  • It is one of the most common used programming lanugages that is used with system/application software, drivers, client-servers, applications and embedded software
  • It has a alot of predefined classes that are data types, which can be instantiated many times
  • Classes create objects at run time. Classes can also be inherited by other classes and thus makes public and protected functionalities by default
  • One top feature from this language is the ability to overload certain operators such as addition
  • Created in 1979 by Bjarne Stroustrup
