Programming Language(C++)

General Information Pros Cons Who uses c++?
  • Highly portable language which is often chosen to be used for multi-device and multi-platform app development
  • Object oriented and uses classes, inheritance, polymorphism, data abstraction and encapsulation
  • Allows exception handling and ability to overload functions
  • Powerful, efficient and fast language
